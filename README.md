<h2>Features</h2>
<ul>
  <li><h3>Responsive Design:</h3>
    The website layout adapts seamlessly to different screen sizes, providing an optimal viewing experience for all users.</li>
  <li><h3> Interactive Animations:</h3>
    Various interactive features have been incorporated using JavaScript, adding an extra touch of delight to the user experience. </li>
  <li><h3> Scroll Animations:</h3>
    As you navigate through the pages, elements come to life with smooth scrolling animations, making the website engaging and visually   appealing.</li>
  <li><h3> Heart Animations: </h3>
    Prepare to be charmed! Keep an eye out for adorable heart animations sprinkled throughout the website, bringing some warmth and love. </li>
  <li><h3> Content Organization:</h3>
    The website is divided into separate HTML pages for easy navigation and to ensure a clear presentation of different sections.</li>
  
</ul>  

I designed the website to be responsive using CSS, ensuring compatibility on various devices, including desktops, laptops, tablets, and smartphones.
